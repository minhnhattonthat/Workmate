# Workmate - Clock-in & Clock-out

## Build

### Installation

Install dependencies.

```bash
yarn install
```

#### IOS

##### CocoaPods

Install pods.

```bash
cd ios
pod install
cd ../
```

Run on iOS

```bash
npx react-native run-ios
```

#### Android

Run on Android

```bash
npx react-native run-android
```

## Testing

```bash
yarn run test
```
